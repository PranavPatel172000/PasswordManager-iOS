# 🔐 Password Manager - SwiftUI iOS App

A modern **iOS Password Manager application** inspired by the native iOS Passwords experience, built using **SwiftUI**.
This app securely stores user credentials using **Keychain Services** and demonstrates strong fundamentals in **security, architecture, and state management**.

---

## 🚀 Features

* 🔐 Secure password storage using Keychain
* ➕ Add and manage credentials (app/website, username, password)
* 👀 View saved passwords in a clean UI
* 🔍 Real-time search functionality for quick access
* 📊 Advanced sorting options
* 🧠 Built with scalable MVVM architecture
* 🎨 Modern and minimal UI using SwiftUI
* 🔒 Data encryption handled by iOS Keychain

---

## 🛠️ Tech Stack

* **SwiftUI** – Declarative UI framework
* **Keychain Services** – Secure data storage
* **MVVM Architecture** – Clean and scalable structure
* **Swift** – Core programming language

---

## 🧠 Architecture

The project follows the **MVVM (Model-View-ViewModel)** pattern:

* **Model** → Represents credential data
* **ViewModel** → Handles business logic, sorting & search logic
* **View** → SwiftUI views for UI rendering

---

## 🔑 Security (Keychain Integration)

* Passwords are stored securely using **Apple Keychain**
* Data is encrypted and protected by the system
* Prevents unauthorized access to sensitive information
* Industry-standard secure storage used in real-world apps
